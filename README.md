# Spring Boot Admin Console (spring-boot-admin-console)
Spring Boot Admin is a community project to manage and monitor Spring Boot applications. It provides a web-based UI to visualize and interact with Spring Boot Actuator endpoints.

**URL:** [Visit APIs.json URL](https://github.com/codecentric/spring-boot-admin)
